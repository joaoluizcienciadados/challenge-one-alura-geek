# Desafio ONE - Alurageek

## Sobre o desafio
Desenvolver o AluraGeek, página de e-commerce.  
O objetivo deste desafio é praticar o CRUD, mais especificamente a parte de Create, Read e Delete (Criação, Leitura e Exclusão).
